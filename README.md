# UnsplashのAPIを利用した写真表示アプリ
<br>

## 新着画像を5枚取得する
毎日更新される画像を5枚取得するようトップページを構成しました。
仕様書を確認しながら新着画像を取得するパラメーターを実行しています。
<br><br>

## デザインフッター
タブではなく、デザイナーがFigmaを用いてオリジナルにデザインした固定フッダーを利用し、デザインを意識した構成で開発しました。
